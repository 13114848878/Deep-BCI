<Cognitive_Task_of_Working_Memory_Program_using_Psychtoolbox>

*Programming Language: MATLAB
*Contact: Sunghee Dong (zpfzpf123@korea.ac.kr)
*Function: We provide a program designed to allow subjects to perform cognitive tasks (default: subtraction, modifiable) based on working memory using the freely distributed psychtoolbox. Through the program, the experimenter can show participants the screen to perform the cognitive task, export the trigger, or input the response using the keyboard.
